Net Ninja - Node JS Tutorial for Beginners

Node.js is:

  • A platform which allows us to run JavaScript on a computer or server
  • Read, delete and update files
  • Easily communicate with a database

Node.js' package ecosystem is npm. npm is the largest ecosystem of open-source libraries.

Why is Node.js so Popular?

  • It uses JavaScript
  • Very fast (runs on the V8 engine and uses non-blocking code)
  • Huge ecosystem of open-source packages (npm)
  • Great for real-time services (like chats)
